# RunTipi app (duplicate layout)

The **canonical** RunTipi app is under **`apps/hundtracker/`** at the repo root (RunTipi expects `apps/<app-id>/`).

This folder (`runtipi-app/hundtracker/`) is a copy for reference. Use **`apps/`** when adding this repo as a RunTipi app store. See the main [README](../README.md) section **Deploy to GitHub Container Registry (GHCR) and RunTipi**.
